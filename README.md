# Data Preprocessing Task

## Dataset
Raw dataset used: Student Performance Dataset

## Steps Performed
1. Loaded raw dataset
2. Handled missing values using mean and mode
3. Encoded categorical features using Label Encoding
4. Standardized numerical features using StandardScaler
5. Split data into training and testing sets (80-20)
6. Saved cleaned dataset

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
